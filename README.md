## Description
Installing TeamPass server on Debian 9

## Requirements
- Debian 9
- MySQL server 5.7.x


## Role Variables

  MySQL_ROOT_ACCOUNT  : root

  MySQL_ROOT_PASSWORD : password

  tp_dbname           : teampass

  tp_dbuser: teampass : teampass

  tp_dbpass: teampass : teampass

  teampass_server_name : teampass.turismo.gov.br
